# Module Sous-pages (admin)

## Fonctionnement en admin
- Affiche automatiquement les sous-pages d’une page parente si le champ "chapeau" n’est pas renseigné.
- Les sous-pages sont gérées dans l’admin WordPress comme des pages classiques.

### Créer une nouvelle sous-page
1. Dans le menu WordPress, cliquez sur "Pages".
2. Cliquez sur "Ajouter" pour créer une nouvelle page.
3. Dans les attributs de la page, sélectionnez la page parente.
4. Remplissez les champs :
	- **Titre** : le titre de la sous-page.
	- **Contenu** : le texte principal.
	- **Image mise en avant** : ajoutez une image pour la sous-page.
5. Cliquez sur "Publier" pour enregistrer la sous-page.

### Modifier ou supprimer une sous-page
1. Dans la liste des pages, cliquez sur le titre de la sous-page à modifier.
2. Modifiez les champs souhaités puis cliquez sur "Mettre à jour".
3. Pour supprimer une sous-page, cliquez sur "Déplacer dans la corbeille".

### Organisation et affichage
- Les sous-pages sont affichées automatiquement sous la page parente si le champ "chapeau" n’est pas renseigné.
- Les images mises en avant et les liens sont générés automatiquement.

## Conseils
- Pour afficher ou masquer le module, renseignez ou non le champ "chapeau" sur la page parente.
- Ajoutez des images mises en avant pour chaque sous-page.
