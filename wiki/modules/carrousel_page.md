# Module Carrousel de sous-pages

## Fonctionnement en admin
- Utilisation via le shortcode `[slider_carousel_pages parent_id="123"]`.
- Affiche les pages enfants d’une page parente sous forme de slider/carrousel.

### Créer une nouvelle sous-page pour le carrousel
1. Dans le menu WordPress, cliquez sur "Pages".
2. Cliquez sur "Ajouter" pour créer une nouvelle page.
3. Dans les attributs de la page, sélectionnez la page parente (celle utilisée dans le shortcode).
4. Remplissez les champs :
   - **Titre** : le titre de la sous-page.
   - **Contenu** : le texte principal.
   - **Image mise en avant** : ajoutez une image pour la sous-page.
5. Cliquez sur "Publier" pour enregistrer la sous-page.

### Modifier ou supprimer une sous-page
1. Dans la liste des pages, cliquez sur le titre de la sous-page à modifier.
2. Modifiez les champs souhaités puis cliquez sur "Mettre à jour".
3. Pour supprimer une sous-page, cliquez sur "Déplacer dans la corbeille".

### Organisation et affichage
- Les sous-pages sont affichées dans le carrousel selon l’ordre défini et les paramètres du shortcode.
- Les titres, extraits et images des sous-pages sont affichés automatiquement.
- Si aucun résultat, vérifiez l’ID de la page parente et la publication des sous-pages.

## Conseils
- Utilisez des images mises en avant pour chaque sous-page pour un rendu optimal.
- Testez le shortcode sur une page dédiée pour valider l’affichage.
