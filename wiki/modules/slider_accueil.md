# Module Slider accueil (admin)

## Fonctionnement en admin
- Module basé sur le Custom Post Type `slider`.
- Les slides sont gérées dans le menu "Slider page d’accueil" de l’admin WordPress.

### Créer une nouvelle slide
1. Dans le menu WordPress, cliquez sur "Slider page d’accueil".
2. Cliquez sur "Ajouter une slide".
3. Remplissez les champs :
   - **Titre** : le titre de la slide (affiché ou non selon le template).
   - **Image** : ajoutez une image mise en avant pour la slide.
   - **Vidéo** (optionnel) : collez le code iframe YouTube si le template le permet.
   - **Bouton** : renseignez le texte du bouton et l’URL de redirection (si disponible dans le template ou via ACF).
4. Cliquez sur "Publier" pour enregistrer la slide.

### Modifier ou supprimer une slide
1. Dans la liste des slides, cliquez sur le titre de la slide à modifier.
2. Modifiez les champs souhaités puis cliquez sur "Mettre à jour".
3. Pour supprimer une slide, cliquez sur "Déplacer dans la corbeille".

### Organisation et affichage
- Les slides sont affichées dans l’ordre défini (menu_order). Vous pouvez réorganiser l’ordre via l’interface d’administration.
- Pour afficher le slider en front, utilisez le shortcode ou le template dédié selon la documentation du thème.

## Conseils
- Ajoutez des images de qualité pour chaque slide.
- Vérifiez l’ordre des slides pour le rendu sur la page d’accueil.
