# Module Renommage projet/réalisation

## Fonctionnement en admin
- Ce module modifie le nom et le slug du Custom Post Type "project" en "Réalisation".
- Les catégories et tags sont aussi renommés ("type-de-realisation", "filtre-de-realisation").
- Les réalisations sont accessibles dans le menu "Réalisation" de l’admin WordPress.

### Créer une nouvelle réalisation
1. Dans le menu WordPress, cliquez sur "Réalisation".
2. Cliquez sur "Ajouter une réalisation".
3. Remplissez les champs :
	- **Titre** : le titre de la réalisation.
	- **Description** : le contenu principal.
	- **Image** : ajoutez une image mise en avant.
	- **Catégories** : sélectionnez le type de réalisation.
	- **Tags** : ajoutez des filtres de réalisation.
4. Cliquez sur "Publier" pour enregistrer la réalisation.

### Modifier ou supprimer une réalisation
1. Dans la liste, cliquez sur le titre de la réalisation à modifier.
2. Modifiez les champs souhaités puis cliquez sur "Mettre à jour".
3. Pour supprimer une réalisation, cliquez sur "Déplacer dans la corbeille".

### Organisation et affichage
- Les réalisations sont affichées dans l’ordre défini. Vous pouvez réorganiser l’ordre via l’interface d’administration.
- Les catégories et tags permettent de filtrer et organiser les réalisations sur le site.

## Conseils
- Vérifiez que les permaliens sont bien mis à jour après l’activation du module.
- Utilisez les nouveaux slugs pour les liens internes.
