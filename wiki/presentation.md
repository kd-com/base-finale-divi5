# Présentation générale du thème kd-com

Le thème **kd-com** est un child theme WordPress conçu pour offrir une grande flexibilité et une personnalisation avancée grâce à l’intégration d’ACF, de blocks Gutenberg, de modules front et admin, et d’une gestion complète des styles via SASS.


## Objectifs du thème
- Faciliter la création de pages dynamiques et modulaires
- Permettre aux développeurs d’ajouter facilement des fonctionnalités
- Offrir aux administrateurs une interface claire pour gérer le contenu

## Structure du thème

```
kd-com/
├── acf-json/           # Configuration des champs ACF
├── ajax/               # Scripts AJAX personnalisés
├── assets/             # Ressources diverses (icônes, polices...)
├── blocks/             # Blocks Gutenberg personnalisés
├── css/                # Fichiers CSS compilés
├── img/                # Images et icônes
├── includes/           # Fonctions PHP additionnelles
├── js/                 # Scripts JavaScript
├── module_admin/       # Modules pour l’administration
├── module_front/       # Modules pour le front-office
├── sass/               # Sources SASS pour les styles
├── functions.php       # Fonctions principales du thème
├── style.css           # Feuille de style principale
├── theme.json          # Configuration du thème
└── wiki/               # Documentation du thème
```

## Fonctionnalités principales
- Blocks ACF et Gutenberg personnalisés
- Modules réutilisables pour le front et l’admin
- Compilation SASS pour une gestion avancée des styles
- Organisation claire des fichiers pour faciliter le développement
