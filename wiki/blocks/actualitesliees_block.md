# Block "Actualités liées"

## Fonctionnement utilisateur

### Ajouter le block sur une page ou un article
1. Ouvrez la page ou l’article à modifier dans l’éditeur Gutenberg.
2. Cliquez sur le bouton “+” pour ajouter un nouveau block.
3. Tapez "Actualités liées" dans la barre de recherche des blocks.
4. Sélectionnez le block "Actualités liées" dans la liste proposée.

### Créer, modifier ou supprimer une actualité liée
1. Les actualités affichées sont issues des pages ou articles sélectionnés comme "parente" dans le block.
2. Pour ajouter une actualité, créez une nouvelle page ou un nouvel article et associez-le à la page parente.
3. Pour modifier une actualité, éditez le contenu ou les champs personnalisés de la page ou de l’article concerné.
4. Pour supprimer une actualité, déplacez la page ou l’article dans la corbeille.

### Organisation et affichage
- Le block affiche les actualités selon les paramètres choisis (nombre de colonnes, ordre, style, couleurs, etc.).
- Vous pouvez réorganiser l’ordre des actualités via l’interface d’administration WordPress.

### Conseils
- Prévisualisez la page pour vérifier le rendu avant de publier.
- Utilisez des titres et descriptions clairs pour chaque actualité afin d’améliorer l’accessibilité et le référencement.
- Si le block ne s’affiche pas correctement, vérifiez que la page parente est bien sélectionnée et que les options sont renseignées.

### Astuce
Vous pouvez dupliquer le block "Actualités liées" pour réutiliser la même configuration sur une autre page. Cliquez sur les options du block (trois points) puis “Dupliquer”.
