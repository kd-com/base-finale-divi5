# Block "Sous-pages"

## Fonctionnement utilisateur

### Ajouter le block sur une page ou un article
1. Ouvrez la page ou l’article à modifier dans l’éditeur Gutenberg.
2. Cliquez sur le bouton “+” pour ajouter un nouveau block.
3. Tapez "Sous-pages" dans la barre de recherche des blocks.
4. Sélectionnez le block "Sous-pages" dans la liste proposée.

### Créer, modifier ou supprimer une sous-page
1. Les sous-pages affichées sont issues des pages associées à la page parente sélectionnée dans le block.
2. Pour ajouter une sous-page, créez une nouvelle page et sélectionnez la page parente dans les attributs de la page.
3. Pour modifier une sous-page, éditez le contenu ou les champs personnalisés de la page concernée.
4. Pour supprimer une sous-page, déplacez la page dans la corbeille.

### Organisation et affichage
- Le block affiche les sous-pages selon les paramètres choisis (nombre de colonnes, ordre, style, couleurs, etc.).
- Vous pouvez réorganiser l’ordre des sous-pages via l’interface d’administration WordPress.

### Conseils
- Prévisualisez la page pour vérifier le rendu avant de publier.
- Utilisez des titres et descriptions clairs pour chaque sous-page afin d’améliorer l’accessibilité et le référencement.
- Si le block ne s’affiche pas correctement, vérifiez que la page parente est bien sélectionnée et que les options sont renseignées.

### Astuce
Vous pouvez dupliquer le block "Sous-pages" pour réutiliser la même configuration sur une autre page. Cliquez sur les options du block (trois points) puis “Dupliquer”.

---

Ce block facilite la navigation entre les pages d’un même ensemble. Pour toute question ou problème, contactez l’équipe support ou consultez le wiki du thème.
