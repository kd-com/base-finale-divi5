# Block "Videos Grid"

## Fonctionnement utilisateur

### Ajouter le block sur une page ou un article
1. Ouvrez la page ou l’article à modifier dans l’éditeur Gutenberg.
2. Cliquez sur le bouton “+” pour ajouter un nouveau block.
3. Tapez "Videos Grid" dans la barre de recherche des blocks.
4. Sélectionnez le block "Videos Grid" dans la liste proposée.

### Créer, modifier ou supprimer une vidéo affichée
1. Les vidéos affichées sont extraites automatiquement du contenu des articles, pages ou CPT selon la source sélectionnée dans le block.
2. Pour ajouter une vidéo, éditez le contenu du post/page/CPT concerné et ajoutez une URL YouTube (via un block "Intégration" ou "HTML").
3. Pour modifier une vidéo, modifiez l’URL ou le contenu du post/page/CPT.
4. Pour supprimer une vidéo, retirez l’URL du contenu du post/page/CPT.
5. En mode manuel, renseignez la liste des vidéos dans le champ dédié du block.

### Organisation et affichage
- Le block affiche les vidéos selon les paramètres choisis (nombre de colonnes, titre, style, ratio, animation, RGPD, etc.).
- Vous pouvez réorganiser l’ordre des vidéos en modifiant l’ordre des posts/pages/CPT ou la liste manuelle.

### Conseils
- Prévisualisez la page pour vérifier le rendu avant de publier.
- Si le block ne s’affiche pas correctement, vérifiez la source sélectionnée et la présence d’URLs YouTube dans le contenu des posts/pages.

### Astuce
Vous pouvez dupliquer le block "Videos Grid" pour réutiliser la même configuration sur une autre page. Cliquez sur les options du block (trois points) puis “Dupliquer”.
