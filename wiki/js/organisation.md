# Organisation et rôle des fichiers du dossier js

Le dossier `js/` contient les scripts JavaScript utilisés pour les interactions et animations du thème kd-com.

## Structure courante
- `animation-sidebar.js` : gestion des animations de la sidebar
- `aos-divi.js` : animations AOS pour Divi
- `aos-frontend.js` : animations AOS pour le front

## Bonnes pratiques
- Rangez chaque script par fonctionnalité ou composant.
- Documentez le rôle et les dépendances de chaque fichier.
- Utilisez des commentaires pour expliquer les fonctions principales.
